🔧 G-code Recovery
Workflow: (1) Home machine via sender. (2) Fill in the three sections below. (3) Build and send.

The positioning replay block replays moves from the last G0 up to the stop point with the pen kept up — the machine traces the same path in the air to arrive at the exact XY with correct arc history, then puts the pen down. The line where the machine stopped should go in ③ so it is fully drawn with the pen down — the machine may have stopped at the start or mid-line, so it needs to run again.
① Startup G-code
② Positioning replay — paste from the last G0 up to the line before where the machine stopped
All Z-down moves are suppressed so the pen stays up throughout. The machine retraces the path in the air to arrive at the correct position. The last line here should be the last completed command before the stop.
③ Remaining G-code — paste from the line where the machine stopped (inclusive) onwards
Include the line where it stopped — it will be drawn properly with the pen down. If the machine stopped mid-line it will simply re-draw that segment from the beginning which is fine.
